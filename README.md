# DretchroNeo

Retro style game console.

# Parts

Raspberry Pi Pico
[Adafruit Color Display](https://www.adafruit.com/product/4311)
Buttons

# Code Setup

1.  Circuit Python v10.
2.  Circuit Python Bundle for v10.
3.  Copy the following to "lib\\" from the bundle.
    1. "adafruit_bus_device\\"
    2. "adafruit_display_text\\"
    3. "adafruit_st7789.mpy"
4.  Copy "code.py" to the root.
